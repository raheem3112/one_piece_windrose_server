Character Repair

Fixes the known Windrose progression drift where a character crashes the server on join with RewardLevel < CurrentLevel. Runs against a copy of your uploaded save and returns a repaired zip.

  1. Close Windrose. Check Task Manager if needed.
  2. Open %LOCALAPPDATA%\R5\Saved\SaveProfiles\.
  3. Right-click your long-number SteamID folder and create a zip.
  4. Choose that zip below. Do not upload the server world save.
  5. After download, extract the repaired zip back into SaveProfiles and replace the existing files.

Max upload 200 MB. Safe mode only fixes no-spend progression drift. Anything else, the tool refuses instead of guessing.
